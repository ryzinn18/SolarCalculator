Improved Cost Insights

One key insight missing from the results of this tool is estimating of the total investment to have solar installed at your residence. This update wil included results for:

  • The total estimated cost for installing solar based on your energy needs.
  • A breakdown of the Federal Tax Credit and how it affects your total costs.
  • An estimate of how long until your solar array pays for itself.

    • Customize Solar Array Size and Module Cost

    In addition to the new insights discussed above, this update will also allow users more control over their inputs. With respect to the insights regarding total costs, this tool will initially execute it's first 2 steps (Input Validation & Retrieving Solar Data; see Business Logic above) and display the suggested solar array size and estimated solar array cost. It will allow the user to modify these parameters to better fit their needs. For example, in the case that the user wants a larger solar array than suggested, they can modify that parameter before calculating their final results. This functionality will ensure users are getting results that best fit their situation.

    Parallel Processing for Large Tasks

    Currently, the most resource intensive tasks executed by this application is the generation of the graphs displayed for the results. To improve this, the portion of the program that creates these graphs and writes them to their respective S3 buckets will be abstracted and calls will be made in parallel. This will allow the program to create the 2 (and maybe more to come) graphs and .csv file at the same time using asynchronous Lambda calls.
